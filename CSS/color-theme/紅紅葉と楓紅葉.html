典拠と色の構成

平安末期『満佐須計装束抄』に「紅・山吹・黄なる・青き濃き薄き・紅の単」と記される秋の五衣。名は紅でも赤一色ではなく、紅から黄を経て青(緑)へ移ろうグラデーションが要です。(中世武家の『雁衣鈔』では「面赤色・裏濃赤色」と真紅になり、時代で色が異なります。)

典拠についての注記

「楓紅葉」は古典の原典(『満佐須計装束抄』など)には確認できず、近現代の重ね色目集(長崎盛輝『かさねの色目』系統)由来の可能性が高い色目です。配色は「青紅葉」の変種に近く、淡青(緑)から黄・朽葉を経て紅・蘇芳へ至る移ろいとして楓の色づきを表します。

CSS 変数 (Design Tokens)

そのまま開発に貼り付けられる形です。[data-theme] でテーマを、[data-mode="light"] でライト版を切り替えます。Docker / devcontainer 上のフロントにもそのまま読み込めます。


  






  

    この見本について。
    色名と配色は平安末期『満佐須計装束抄』を中心に、綺陽装束研究所・日本服飾史・長崎盛輝『かさねの色目』等の整理を参照しています。
    重ね色目の色は文献・流派・時代で解釈が分かれ、伝統色のRGB/HEX値はいずれも目安です。本テーマの各HEXはWeb表示・可読性のために調整した近似値で、史料の厳密な色そのものではありません。
    
コントラスト比は表示中のモードの実際の色から算出しています。ライト版の副え見出し・差し色は文字用に濃色(secondary-ink / accent-ink)を割り当て、彩りのチップには本来の山吹・紅をそのまま使っています。
