典拠についての注記

平安の重ね色目に「紫陽花」という名の色目は、古典の有職故実書にも標準的な色目集にも確認できません。万葉集に「あぢさゐ」は詠まれますが、装束のかさね色目には採られませんでした。このテーマは典拠のある夏の青紫系色目(露草=縹・薄縹、棟=薄紫・青、二藍など)に寄せた現代の再構成です。「青」は平安では緑を指す語でしたが、ここでは現代の紫陽花のイメージに合わせ青系で構成しています。

花色の背景

装飾花の青は、酸性土壌でアルミニウムがイオン化し色素と結びつくことで生まれます。火山性で酸性土の多い日本では青系が出やすく、藍姫(あいひめ)、伊予の青絣(いよのあおがすり)、シーボルトが記し1959年に六甲で再発見された幻の七段花(しちだんか)などが知られます。色は咲く環境で変わるため、ここでの伝統色対応は典型例です。

花色の背景

中性〜アルカリ性の土ではアルミニウムが吸収されず、色素本来の紅・紅紫が現れます。紅(くれない)は土に左右されず日光で白→ピンク→真紅へ染まる人気種。ほかに白地に紅の覆輪をもつ古品種清澄沢(きよすみさわ)、白から紅へ移る紅てまりなどがあります。色は環境で変わるため、伝統色対応は典型例です。

CSS 変数 (Design Tokens)

そのまま開発に貼り付けられる形です。[data-theme] でテーマ(ajisai-kasane / ao-ai / kurenai-murasaki)を、[data-mode="light"] でライト版を切り替えます。Docker / devcontainer 上のフロントにもそのまま読み込めます。


  






  

    この見本について。
    色名と配色は、平安の夏のかさね色目(露草・棟・二藍など)と、ヤマアジサイ在来品種の花色を日本の伝統色に対応づけたものです。
    重ね色目「紫陽花」は古典の典拠が確認できず、テーマ①は現代の再構成です。花色は土壌・日照・株齢で大きく変わるため、品種と色の対応は典型例で、各HEXはWeb表示・可読性のために調整した目安です。
    
コントラスト比は表示中のモードの実際の色から算出。ライト版の副え見出し・差し色は文字用に濃色(secondary-ink / accent-ink)を割り当て、彩りのチップには鮮やかな色をそのまま使っています。
