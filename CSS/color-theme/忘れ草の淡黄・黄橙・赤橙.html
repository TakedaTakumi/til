朝にひらき、夕べにしぼむ一日花(ひとひばな)。属名 Hemerocallis は「一日の美」の意。 別名を忘れ草といい、身につければ憂いを忘れるという伝えが万葉のころから詠まれてきました。 同じ属のなかで花色は淡黄から赤橙へと移ろいます。その色幅から三つのカラーテーマを仕立て、 宵(ダーク)と朝(ライト)を切り替えられ、主要な配色にはWCAGコントラスト比を表示しています。

典拠についての注記

「淡黄」は色目集にある固有の色名ではなく、淡く咲くユウスゲ(夕菅)の花色を、支子(くちなし)など黄系の伝統色に寄せた現代の配色です。ユウスゲは夕方にひらいて翌朝しぼむ夜咲きの一日花。各HEXは目安で、史料や実際の花の色と厳密には一致しません。

典拠と色の構成

萱草色(かんぞういろ)は萱草(ワスレグサ)の花の色に由来する実在の伝統色で、明るい黄みの橙。福田邦夫『色の名前事典』は「強い赤みの黄」とし、平安期から用いられた色名とします(HEXは目安で #FDA900〜#F8B862 等、出典で差があります)。源氏物語「葵」には喪の「萱草の袴」が見え、忘れ草にちなみ喪・忌色にも用いられました。このテーマは萱草色を軸に、山吹・柑子など黄〜橙の伝統色で構成しています。

典拠と色の構成

ノカンゾウは橙〜橙赤、ヤブカンゾウは八重の濃い橙。萱草の赤橙側を、橙(だいだい)・黄丹(おうに)・緋(ひ)など黄赤〜赤の伝統色で構成しました。黄丹は萱草色と同じ紅花＋クチナシ系統の染料で、皇太子の袍に用いる格高い禁色。緋は茜染めの鮮やかな赤。色名のHEXは目安で、実際の花色は個体差が大きく一致しません。

CSS 変数 (Design Tokens)

そのまま開発に貼り付けられる形です。[data-theme] でテーマ(usuki / ki-daidai / aka-daidai)を、[data-mode="light"] でライト版を切り替えます。Docker / devcontainer 上のフロントにもそのまま読み込めます。


  






  

    この見本について。
    色名と配色は、忘れ草(ワスレグサ属/カンゾウ)の同属各種の花色を日本の伝統色に対応づけたものです。軸となる萱草色は実在の伝統色ですが、「淡黄」「黄橙」「赤橙」というテーマ名は色域を示す呼称で、固有の色目名ではありません。
    なおかさね色目の「萱草」は、季節の美を競う通常の襲色目には含まれず、表裏とも萱草色を用いる喪服の特殊な色目として説明されるもので、平安伝来の季節色目と断定はできません。花色は個体差・環境で大きく変わり、各HEXはWeb表示・可読性のために調整した目安です。
    
コントラスト比は表示中のモードの実際の色から算出。ライト版の副え見出し・差し色は文字用に濃色(secondary-ink / accent-ink)を割り当て、彩りのチップには鮮やかな色をそのまま使っています。忘れ草=憂いを忘れる草というモチーフは『万葉集』(大伴家持ほか)以来詠まれ、中国・嵆康『養生論』の「萱草忘憂」が背景にあります。
